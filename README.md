# IELTS Anger Teacher Bot 🎤😡

A savage Telegram voice bot that criticizes your speaking skills like an angry IELTS examiner.

## Features
- 🎤 Voice message input (English, Russian, Uzbek)
- 🔊 Voice response output (English, Russian)
- 😡 Unhinged, hilarious personality
- 📝 Text fallback for Uzbek

## Quick Start

### 1. Get API Keys
- **Telegram Bot Token**: From @BotFather on Telegram
- **Deepgram API Key**: $200 free credit at deepgram.com
- **Google TTS API Key**: 1M chars/month free at cloud.google.com
- **OpenAI API Key**: Optional, for better transcription

### 2. Deploy on Render (Free)

1. Push this code to GitHub
2. Create a new Web Service on Render
3. Connect your repository
4. Add environment variables in Render dashboard
5. Deploy!

### 3. Environment Variables

```env
TELEGRAM_TOKEN=your_token
DEEPGRAM_API_KEY=your_key
GOOGLE_TTS_API_KEY=your_key
OPENAI_API_KEY=your_key  # Optional
